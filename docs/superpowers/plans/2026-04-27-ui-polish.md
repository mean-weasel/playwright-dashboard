# UI Polish Bundle Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add live thumbnails to the menubar popover, drag-to-reorder in the session grid, and essential keyboard shortcuts (Escape to go back, single-click to open).

**Architecture:** All three features are modifications to existing SwiftUI views — no new files. Thumbnails reuse the existing `NSImage(data:)` pattern from `SessionCard`. Drag reorder completes the already-wired `.draggable()` with a `.dropDestination()` handler. Keyboard shortcuts use SwiftUI's `.keyboardShortcut()` modifier.

**Tech Stack:** SwiftUI, SwiftData (@Model observation), NSImage.

- [ ] **Step 3: Add the `reorder` helper function**

Add this private method at the bottom of the `SessionGrid` struct, after the `filteredSessions` computed property (after line 138):

```swift
  private func reorder(sourceId: String, targetId: String) -> Bool {
    guard sourceId != targetId else { return false }
    guard let source = appState.sessions.first(where: { $0.sessionId == sourceId }),
      let target = appState.sessions.first(where: { $0.sessionId == targetId })
    else { return false }

    let temp = source.gridOrder
    source.gridOrder = target.gridOrder
    target.gridOrder = temp
    return true
  }
```
