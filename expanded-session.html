Interaction flow — how views connect

Navigation

  • Menubar click > popover (compact list)
  • Popover "Open Dashboard" > full NSWindow grid
  • Grid double-click card > expanded view (this page)
  • "Detach Window" > floating PiP mini-window
  • Esc or back arrow > returns to grid
  • All views share one process, instant transitions

Expanded view features

  • Live CDP screencast (real-time page updates)
  • Right panel tabs: Info, Tabs, Console, Network
  • Console shows live browser logs (filterable)
  • Screencast bar: screenshot, record GIF, refresh
  • "Open CDP" opens Chrome DevTools Protocol inspector
  • Click any browser tab to switch active view
  • Rename inline by clicking the name
  • PiP windows are always-on-top, resizable
